# Proyecto VR – Interacciones Básicas en Unity

## Alumna
Alma Janet Gómez Mejía

## Descripción del proyecto
En este proyecto desarrollé una escena de Realidad Virtual (VR) utilizando Unity y el XR Interaction Toolkit, con el objetivo de implementar las interacciones básicas vistas en clase.  
La aplicación permite interactuar con objetos dentro de un entorno virtual mediante manos virtuales, realizando acciones como agarrar, soltar y detectar colisiones, logrando una experiencia inmersiva en realidad virtual.

El desarrollo del proyecto se realizó de manera individual, siguiendo los procedimientos enseñados en clase y adaptándolos a mi propio trabajo.

## Funcionalidades implementadas
Durante el desarrollo del proyecto implementé las siguientes funcionalidades:

- Configuración de un proyecto de Realidad Virtual en Unity.
- Integración y configuración del XR Interaction Toolkit.
- Configuración del XR Origin (XR Rig).
- Uso de controladores basados en acciones (Action Based Controllers).
- Implementación de manos virtuales animadas mediante el paquete **Oculus Hands**.
- Configuración de objetos interactivos que permiten:
  - Agarrar objetos.
  - Soltar objetos de forma natural.
  - Detectar colisiones físicas.
- Creación de un escenario con un plano para desplazamiento y una mesa para la interacción con los objetos.

## Instrucciones para ejecutar el proyecto
Para ejecutar correctamente el proyecto se deben seguir los siguientes pasos:

1. Abrir **Unity Hub**.
2. Seleccionar la versión de Unity utilizada para el desarrollo del proyecto.
3. Abrir la carpeta del proyecto desde Unity Hub.
4. Verificar que el paquete **XR Interaction Toolkit** esté instalado.
5. Confirmar que el **Input System** esté configurado correctamente.
6. Ejecutar la escena principal desde Unity.
7.  Conectar un visor Oculus Quest mediante Meta Quest Link para probar la experiencia en Realidad Virtual.
